Local-first link routing

Every link, in the right browser profile.

TabFlow watches links before they become messy tabs. Pair the desktop app with the browser extension, link each profile once, and let work, personal, client, and research links open exactly where they belong.

Profile-aware

Rules point to a browser and a profile.

The current app discovers installed browsers, scans profiles, and only lets linked extension profiles become routing destinations. That means a “docs” link can land in Chrome Work while another opens in Firefox Personal without manual copy-paste.

Install path

Desktop app first, extension per profile.

  1. App

    Download the latest TabFlow App build for your platform.

  2. Extension

    Install TabFlow in the browser profile you want to route into.

  3. Link

    Open the app, choose a detected profile, and complete the extension link.

  4. Rule

    Create URL rules that target linked browser profiles.
